项目一：Olist 电商平台经营分析项目

基于 Kaggle Olist 多表电商数据，使用 MySQL / Excel / Power BI 完成数据清洗、指标建设、经营分析看板和业务结论输出。

16 张 ADS 指标表
3 页 Power BI 看板
5 份 SQL 脚本

核心指标

GMV、订单数、AOV、运费占比、延迟送达率、差评率、复购率、RFM 用户分层、支付方式占比、地区分布、品类销售排行。

核心结论

  • 平台复购率约 3%，用户以一次性购买为主，存在用户留存和复购提升空间。
  • 延迟送达订单评分更低，差评率更高，物流履约体验对用户评价有明显影响。
  • 不同品类和地区的销售贡献差异明显，可结合品类、地区和用户分层进行精细化运营。

项目二：电商用户行为分析项目

使用 Python / Pandas 完成用户行为日志数据清洗，分析浏览、加购、收藏、购买等用户行为路径，并使用 Power BI 展示用户行为漏斗和活跃趋势。

核心分析方向

  • 用户行为转化漏斗：浏览、加购、购买等行为转化情况。
  • 24 小时用户活跃趋势：识别用户活跃高峰时段。
  • 商品类别购买排行：分析不同品类的购买表现。
  • 结合行为转化情况，输出用户转化效率和运营优化建议。
